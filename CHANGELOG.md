# Hope Devlog Skin - Update History (Cosmic & Nature Theme)

## [2026-06-29] V9.0.0 "High-End UX & 3D Interactive Hero"
- **3D WebGL 프리미엄 히어로 적용**: Vanta.js 기반의 3D Net 파티클 효과를 상단 헤더에 적용. 화면 비율(1.5x)과 넓은 간격(Spacing) 설정을 통해 웅장하고 사이버틱한 분위기 연출.
- **집중 읽기 모드 (Zen Mode) 도입**: 글에만 온전히 집중할 수 있도록 사이드바와 상하단 UI 요소를 모두 숨기는 전역 토글 버튼(우측 하단 FAB 내 눈 모양 아이콘) 추가.
- **Mac 스타일 코드 블록 (VS Code 테마)**: `pre > code` 영역을 Mac OS 터미널 스타일(3색 제어 버튼, 언어 뱃지, Copy 버튼)로 리뉴얼하여 개발자 블로그로서의 정체성 강화.
- **가상 SPA(Pseudo-SPA) 트랜지션 추가**: 티스토리 애드센스 및 댓글 플러그인 충돌을 피하면서 부드러운 페이지 전환(Fade In/Out)을 구현하여 앱 라이크(App-like) 퍼포먼스 달성.
- **메인 대시보드 3D Tilt 효과 프리미엄화**: 마우스 호버 시 회전 각도 상향 및 `mix-blend-mode: overlay` 다중 그라데이션을 적용해 입체적인 포켓몬 홀로그램/Apple TV 카드 반사 효과 부여.

## [2026-06-29] V8.3 "Full-Width Mega Menu & Custom Category Thumbnails"
- **풀 위드(Full-Width) 메가 메뉴 적용**: 상단 헤더의 드롭다운 카테고리 메뉴가 개별 항목 기준이 아닌 전체 헤더 가로폭 기준으로 웅장하게 열리도록 설계 변경. 여백과 간격을 2배 이상 넓혀 시각적인 개방감을 확보하고, 글자 수가 긴 서브 카테고리(예: 아키텍처)도 잘림 없이 온전히 표시되도록 그리드 구조 최적화.
- **인기 카테고리 썸네일 커스텀 이미지 지원**: 인기 카테고리 카드(Tech Core Card) 디자인을 사용자가 요청한 깔끔한 세로형 갤러리 디자인으로 전면 교체. 기존에 하드코딩된 SVG 아이콘을 제거하고, 티스토리 스킨 파일 업로드를 통해 사용자 정의 이미지(`./images/카테고리명.png`)를 직접 매핑할 수 있도록 기능 개선.
- **CSS 충돌 및 레이아웃 버그 픽스**: 알약(Pill) 형태로 찌그러지던 인기 카테고리 카드의 CSS 클래스 충돌(`.category-grid` 중복 사용)을 해소하기 위해 독립적인 `.tech-core-grid` 클래스 분리 및 재정의.

## [2026-06-29] V8.2 "Architecture & Security Refactoring"
- **모바일 햄버거 메뉴 애니메이션 고도화 (3D Push & Glassmorphism)**: 모바일 환경에서 햄버거 메뉴를 열 때, 메인 블로그 화면이 뒤로 살짝 밀려나는(3D Push) 입체적인 모션과 함께 반투명한 글래스모피즘(Frosted Glass) 질감의 사이드바가 부드럽게 등장하도록 모바일 UX를 앱 수준으로 고도화.
- **거대 레거시 스크립트 모듈화**: 약 900줄에 달하던 `_legacy.js`를 폐기하고, 최신 프론트엔드 아키텍처에 맞게 `ui.js`, `sidebar.js`, `menu.js`, `post.js`, `effects.js` 모듈로 기능별 완전 분리 및 `main.js`에서 통합 관리.
- **CSS 디자인 토큰 중앙화**: 흩어져 있던 다크/라이트 모드 컬러, 여백, 그림자 설정 등을 `_variables.css`로 통합 분리하여 디자인 시스템 기반 확립.
- **XSS 방어 완벽 적용**: 목차(TOC) 스크롤 스파이, 카테고리 렌더링, 메가 메뉴 등에서 발생할 수 있는 잠재적 스크립트 인젝션(XSS) 취약점을 제거하기 위해 `textContent` 및 `insertAdjacentHTML` 방식으로 렌더링 구조 전면 재설계.
- **퍼포먼스 향상**: 본문 내 대형 이미지 렌더링 최적화를 위해 동적 Lazy Loading 및 Async Decoding 속성 일괄 부여 로직 추가.

## [2026-06-29] V8.1 "UI/UX & Security Refactoring"
- **모바일 사이드바 아코디언 메뉴**: 모바일 햄버거 메뉴 및 우측 사이드바(Drawer)의 카테고리 목록을 아코디언(Accordion) 형태로 개편하여 부드러운 토글 애니메이션 및 서브 카테고리 지원.
- **앱 라이크 스크롤 바 숨김**: 매거진 가로 스크롤 레이아웃에 `::-webkit-scrollbar` 숨김 처리를 적용하여 모바일 앱과 같은 미려한 스와이프 경험 제공.
- **스무스 스크롤링 전역 적용**: 목차(TOC) 클릭이나 내부 앵커 이동 시 부드럽게 스크롤 되도록 `scroll-behavior: smooth` 적용.
- **XSS 방어 보안 리팩토링**: 자바스크립트 DOM 파싱 구간에서 `.innerHTML` 대신 `.textContent`를 사용하도록 구조를 변경하여 보안성과 성능 향상.

## [2026-06-29] V8.0 "Premium Mega Menu & Horizontal Cards"
- **메가 메뉴 다이어그램 개편**: 기존 드롭다운 방식을 벗어나, 마우스 호버 시 글래스모피즘(Glassmorphism) 기반의 넓은 패널이 열리도록 디자인 고도화. 중/소분류 카테고리를 다단(Multi-column) 그리드로 도식화하여 렌더링하고, 포스팅 갯수를 가시성이 뛰어난 뱃지 UI로 강화.
- **섹션 헤더 프리미엄 통합**: 모든 하단/메인 카테고리의 헤더 텍스트 디자인을 왼쪽 정렬 + 시안색(Cyan) 네온 바 형태로 통일하여 강렬한 사이버펑크 룩으로 일관성 유지.
- **가로 스크롤 매거진 레이아웃**: 메인 화면의 카드 레이아웃이 세로로 무한정 늘어나는 문제를 해결하기 위해, PC 기준 정확히 3개씩 배치되도록 너비를 자동 계산(`calc`)하고 넘치는 카드는 스크롤 스냅(Scroll-snap)으로 부드럽게 스와이프 되도록 구조 변경.
- **로컬 Preview 최적화**: `preview.html` 렌더링 시 빌드 결과물 없이도 실시간 모듈 변경(`main.js`)을 확인할 수 있도록 `sync-html.js` 개발 환경 로직 개선. 불필요한 레거시 인기 포스팅 코드(`fetchPopularPosts`) 및 DOM 트리 완전 삭제.

## [2026-06-26] V7.1 "Premium Micro-interactions" 대시보드 디테일 향상
- **대시보드 통계 카드 강화**: 투명한 유리 질감(Glassmorphism)과 화면에 진입 시 숫자가 차오르는 카운트업(Count-up) 애니메이션 적용.
- **매거진 그리드 3D 호버**: 최신글 썸네일 카드에 마우스 방향을 따라 입체적으로 기울어지며 빛이 반사되는 3D 틸트(Tilt & Glare) 효과 추가.
- **인터랙티브 히어로 캔버스**: 마우스 움직임을 따라 별빛 입자들이 물리적으로 흩날리며 반응하는 파티클 캔버스 효과 적용.
- **카테고리 네온 글로우**: 기존 사각형 버튼을 모던한 알약(Pill) 형태로 다듬고 사이버틱한 네온 글로우 호버 디자인 추가.
- **버그 수정**: 다크/라이트 모드 버튼 토글이 간헐적으로 즉시 취소되던 현상 해결 (스크립트 중복 호출 제거).
## [2026-06-25] V7.0 "Cosmic Dashboard" 대규모 UI/UX 릴리즈
- **전역 유성우 애니메이션**: 다크 모드 활성화 시 전체 화면에 사선으로 떨어지는 유성우(Meteor Shower) 이펙트 적용.
- **테크 칩 카테고리 & 레이더 스캔**: 밋밋했던 인기 카테고리를 '글래스모피즘 테크 칩'으로 변경하고, 빈 공간(Empty State)을 '홀로그램 레이더 스캔' 애니메이션으로 개조.
- **코드 블록 및 목차 리뉴얼**: Mac 터미널 스타일의 네온 Copy 버튼 추가 및 우측 하단 목차(TOC) 버튼을 스스로 그려지는 손그림 별(Star) 애니메이션으로 교체.
- **라이트 모드 최적화**: 밝은 모드에서도 가독성이 유지되도록 빛바램 없는 고대비(High-contrast) CSS 오버라이딩 적용.
## [2026-06-25] 사이드바 우주 테마 연동 및 디자인 시스템 개편
