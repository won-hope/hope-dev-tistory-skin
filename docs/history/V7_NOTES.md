# HOPEDEV Skin V7.0 "Cosmic Dashboard" Update Notes

## 🌌 Overview
V7.0은 블로그 스킨을 넘어, 하나의 완벽한 **사이버네틱 우주 정거장 대시보드(Cosmic Space Station Dashboard)**로 탈바꿈시킨 대규모 리뉴얼입니다. 단순한 텍스트 기반 정보 제공을 넘어, 우주 공간의 몰입감을 주는 인터랙티브한 시각적 경험(홀로그램, 레이더, 유성우)을 전역에 적용했습니다.

## ✨ Key Features & Improvements

### 1. 전역 유성우 시스템 (Global Meteor Shower)
- **다크 모드(Dark Mode) 전용 이펙트:** 다크 모드를 활성화하는 순간, 블로그 화면 최하단(z-index: 1) 전역에서 은은하게 빛나는 유성우가 대각선으로 쏟아져 내립니다.
- 눈에 피로를 주지 않고 본문 가독성을 해치지 않도록 투명도와 그라데이션을 세밀하게 조율했습니다.

### 2. 홀로그램 데이터 코어 카테고리 칩 (Data Core Chip)
- 메인 화면의 '인기 카테고리' 카드 영역을 전면 재설계했습니다.
- 다크 글래스모피즘(반투명 블랙 유리) 위로 사이안(Cyan)과 퍼플(Purple) 색상의 빛이 은은하게 퍼져 나오며, `[ SYS / 카테고리명 ]`과 같이 터미널 로그 형태의 폰트로 출력됩니다.
- 포스팅 개수를 우주선 에너지 바(Progress Energy Bar) 형태로 시각화하여 정보의 양을 동적으로 보여줍니다.

### 3. 레이더 스캐닝 빈 상태 화면 (Radar Scan Empty State)
- 특정 카테고리에 글이 없을 때 나오는 밋밋한 텍스트를 **홀로그램 레이더 화면**으로 교체했습니다.
- 위아래로 움직이는 레이저 스캐너 애니메이션과 함께 `[ NO DATA SIGNAL ]`이라는 경고 문구가 깜빡이며, 빈 공간도 하나의 훌륭한 UI 위젯으로 활용했습니다.

### 4. 소스 코드 복사 기능 (Mac Terminal Code Block)
- 개발자 블로그의 필수 요소! 본문 내 소스 코드(`pre > code`) 블록에 MacOS 터미널 스타일의 헤더(빨강/노랑/초록 닷)를 추가했습니다.
- 우측 상단에 배치된 사이버네틱 네온 **[Copy]** 버튼을 클릭하면 `Copied!`라는 애니메이션과 함께 코드가 클립보드에 자동 복사됩니다.

### 5. 핸드 드로잉 별 목차 버튼 (Animated Star TOC)
- 우측 하단의 목차(TOC) 토글 버튼을 밋밋한 햄버거 메뉴(≡)에서 **손그림 느낌의 별(Star) 모양 SVG**로 변경했습니다.
- 가만히 두어도 스스로 선(Stroke)이 스르륵 그려지며 별 모양을 완성하고 네온 빛으로 채워지는 **자동 드로잉 애니메이션**을 부여했습니다.

## 🛠️ Technical Details
- CSS `stroke-dasharray`, `stroke-dashoffset`을 활용한 고급 SVG 드로잉 애니메이션 적용.
- `DOMParser`를 활용하여 티스토리 카테고리 RSS/HTML 데이터를 백그라운드에서 Fetch 해오는 자바스크립트 로직 최적화.
- 다크모드/라이트모드 간 배경색 반전에 따른 가독성 보정(Light Mode Overrides) 완벽 적용.
