# HOPEDEV Tistory Skin V3 Update Notes

## V3에서 만든 정적 기능
- **메가 메뉴**: 데스크탑 헤더에 추가된 다단 구성 탐색 메뉴. 마우스 클릭 및 키보드 Enter/Space 조작 지원. ESC 또는 외부 클릭 시 닫힘.
- **검색 오버레이**: `Search` 버튼 또는 `Ctrl+K`/`Cmd+K` 로 전개되는 전체 화면 검색 패널. 정적 데이터 기반 제목, 요약, 카테고리, 태그 검색 렌더링.
- **전체 글 페이지 (All Articles)**: `preview/category.html`. 카테고리별 동적 필터링을 지원하는 정적 글 아카이브 페이지. 
- **태그 페이지**: `preview/tags.html`. 사용 빈도수에 따라 5단계(1~5)로 폰트 크기가 변화하는 태그 클라우드 페이지.
- **방명록 페이지**: `preview/guestbook.html`. 피드백과 안부를 남기는 정적인 폼 및 예시 스레드 페이지.
- **구독 디자인 슬롯**: 본문 및 홈 화면 최하단에 배치된 "Follow HOPEDEV" 구독 안내 카드 및 토스트 UI 연동 (실제 동작은 치환 예정).

## 티스토리 실제 변환 시 연결할 항목
- **검색**: 검색 결과 리스트 등은 `<s_search>` 영역과 입력 치환자 `[##_search_name_##]` 등으로 교체해야 함. 현재의 V3 정적 JS 검색 로직은 제거하고 티스토리 네이티브 검색 결과 페이지(또는 AJAX 검색 렌더링)로 연결.
- **태그**: 태그 클라우드는 `<s_tag>` 및 `<s_tag_rep>`로 교체하여 루프 렌더링. URL은 `[##_tag_link_##]` 사용.
- **방명록**: `<s_guest>` 영역과 티스토리 기본 방명록 입력 폼/리스트 치환자로 교체 예정.
- **태그 URL**: `/tag` 경로로 연결.
- **방명록 URL**: `/guestbook` 경로로 연결.
- **구독**: 실제 티스토리 네이티브 구독 버튼 스크립트 클래스와 DOM 노드로 치환(또는 `T.config.ROLE` 을 통한 직접 추가)하여 작동하도록 설정.

## 현재 V3에서 의도적으로 구현하지 않은 것
- **실제 구독 상태 저장**: 구독 상태를 저장하거나 서버와 통신하는 기능 (토스트로 갈음됨).
- **실제 방명록 등록**: 폼 내용을 서버에 전송하는 기능 (마찬가지로 토스트 알림으로 갈음됨).
- **서버 검색**: 실제 API나 DB를 통한 백엔드 검색.
- **외부 블로그 팔로우 기능**: 타 티스토리 블로그 연동 및 최신글 피드 뷰.
