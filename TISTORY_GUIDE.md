# HOPEDEV 스킨 티스토리 적용 가이드 🚀

이 문서는 새롭게 리팩토링된 **HOPEDEV Editorial Cubism 스킨(V6.9)**을 여러분의 티스토리 블로그에 안전하고 깔끔하게 적용하는 방법을 안내합니다.

---

## 1. 스킨 파일 준비하기

프로젝트 루트 폴더(`c:\hope-dev-tistory-skin`)에 있는 `refactor.bat`을 실행하셨다면, 현재 폴더 안에는 오직 배포에 필요한 스킨 파일들만 깔끔하게 남아있습니다.

*   `skin.html`
*   `style.css`
*   `index.xml`
*   `images/` 폴더 전체

이 파일들이 스킨 등록에 필요한 **전부**입니다!

---

## 2. 티스토리 스킨 등록 메뉴 접속

1.  내 티스토리 블로그의 **관리자 페이지**로 접속합니다. (`내블로그주소.tistory.com/manage`)
2.  좌측 메뉴 패널에서 **[꾸미기] > [스킨 변경]** 을 클릭합니다.
3.  우측 상단의 **[스킨 등록+]** 버튼을 클릭합니다.

---

## 3. 스킨 파일 업로드하기

스킨 등록 화면에서 2번에 걸쳐 파일을 업로드해야 합니다. 

### 첫 번째: 기본 파일 업로드
1.  **[추가]** 버튼을 클릭합니다.
2.  이 프로젝트 폴더 안에 있는 아래 3개의 파일을 다중 선택하여 엽니다.
    *   `skin.html`
    *   `style.css`
    *   `index.xml`
3.  파일이 목록에 추가된 것을 확인합니다.

### 두 번째: images 폴더 내의 에셋 업로드
> [!IMPORTANT]
> 티스토리는 하위 폴더 구조를 인식하지 못합니다. 따라서 `images` 폴더 전체를 통째로 업로드하는 것이 아니라, **`images` 폴더 안으로 들어가서 그 안의 파일들을 모두 선택**하여 업로드해야 합니다.

1.  다시 한번 **[추가]** 버튼을 클릭합니다.
2.  `images` 폴더 **안으로 진입**합니다.
3.  폴더 안의 **모든 파일(`script.js`, 이미지 파일 등 전체)** 을 선택하여 엽니다.
4.  업로드된 파일 목록의 경로에 `images/script.js` 처럼 `images/` 가 잘 붙어있는지 확인합니다.

---

## 4. 스킨 저장 및 적용

1.  파일 업로드가 끝났다면 우측 상단의 **[저장]** 버튼을 클릭합니다.
2.  스킨의 이름을 입력하라는 창이 뜨면, 구분하기 쉽게 **`HOPEDEV_V6.9`** 처럼 버전명과 함께 입력하고 확인을 누릅니다.
3.  다시 **[스킨 변경]** 화면으로 돌아옵니다.
4.  상단의 **[스킨 보관함]** 탭을 클릭하면 방금 저장한 스킨이 보입니다.
5.  해당 스킨에 마우스를 올리고 **[적용]** 을 클릭하면 블로그에 새 스킨이 완벽하게 반영됩니다! 🎉

---

## 5. 최신 스킨 설정 (선택 사항)

관리자 페이지의 **[꾸미기] > [스킨 편집]** 으로 들어가시면 새롭게 추가된 강력한 기능들을 마우스 클릭만으로 쉽게 제어할 수 있습니다.

*   **포인트 컬러 설정**: 블루, 그린, 퍼플 등 원하는 브랜드 컬러로 즉시 변경 가능
*   **날씨 테마 사용 여부**: 접속자의 날씨(맑음, 비, 눈)에 따라 스킨 테마가 바뀌는 감성적인 모드 on/off
*   **히어로 문구 변경**: 코드 수정 없이 타이핑 텍스트 및 메인 문구 즉각 수정 가능
