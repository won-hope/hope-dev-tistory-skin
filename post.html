프론트엔드 아키텍처와 지속 가능한 설계

Hope 2026. 06. 29.

이 포스트는 Zen Mode와 프리미엄 코드 블록 테마를 테스트하기 위한 로컬 전용 포스트입니다.

아래는 자바스크립트 코드 스니펫 예시입니다:


function calculateZenLevel(focus, distractions) {
  if (distractions > 0) {
    console.warn('Distractions detected!');
    return focus / distractions;
  }
  return focus * 100; // Total Zen
}

CSS 코드 스니펫:


body.zen-mode-active {
  background: #000;
  color: #fff;
}

본문 내용이 충분히 길어야 스크롤 시 사이드바 및 FAB 버튼을 테스트하기 편합니다.
